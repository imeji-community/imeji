Publish Your Scientific Media Data


imeji creates citable research assets by describing, enriching, sharing, exposing and linking data.

This imeji instance contains test collections for the imeji community. The imeji community aims to sustain an active open source software developer community around the imeji software. [more on the imeji community]

The imeji software is an open-source, community driven development. Based on the eSciDoc infrastructure, imeji offers a dedicated metadata management by supporting a non- restrictive metadata schema definition, as simple as liked or as complex as needed. [more on the imeji software]

    Publish Your Scientific Media Data


    imeji erzeugt zitierfähige Forschungsobjekte durch das Beschreiben, Anreichern, Teilen, Bereitstellen und Verlinken von Daten.

    Diese imeji Instanz beinhaltet Testsammlungen für die imeji Community. Die imeji Community zielt darauf ab eine aktive Entwicklungsgemeinschaft rund um die imeji Software zu vereinen. [mehr zur imeji Community]

    imeji ist eine Open-Source Software, welche von der imeji Community entwickelt wird. Sie basiert auf der eSciDoc Infrastruktur und unterstützt die beliebige Definition von Metadaten Schemata wodurch ein dediziertes Metadaten Management ermöglicht wird, so einfach wie möglich und so komplex wie nötig.     マルチメディア科学データの提供、配信


    Imejiは引用可能な研究データ資料集を生成します。マルチメディアデータに説明を付与したり、意味を追加したり、リンクを張るなどして、提供データの価値を高めることができます

    Imejiサービスは、コミュニティ向けのテスト用コレクションを含んでいます。Imejiコミュニティは、活発な開発で持続的なコミュニティとなるよう開発しています。 [Imejiコミュニティの説明]

    Imejiソフトウェアはオープンソースソフトウェアとして提供され、コミュニティ主導により開発している。eSciDocリポジトリ基盤ソフトウェアを基礎として構築されたImejiは、自由なメタデータスキーマをサポートし、拡張的なメタデータ管理を可能としています。ニーズに応じてシンプルなメタデータや複雑なメタデータを構成することができます。 [Imejiソフトウェアの説明]
